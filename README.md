## ðŸ”§ Customization

### Colors
Update gradient colors in components by changing Tailwind classes:
- `from-rose-500 to-orange-500`
- `from-amber-500 to-orange-500`

### Images
Replace Unsplash URLs with your own images in component files.

### Carousel Settings
Modify Swiper configuration in `FeaturedFundraisers.jsx`:
- `autoplay.delay`: Change slide duration
- `breakpoints`: Adjust responsive behavior
- `slidesPerView`: Number of visible slides
